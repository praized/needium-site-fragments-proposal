Fragment Headers Open to suggestions, could be prettier in markdown.

Fragment headers consist of a title and subtitle. These are represented with the <title> and <subtitle> wrappers. The wrappers can contain any valid markdown.

This Markdown

Should Generate

<title>My title</title>
<subtitle>My Subtitle<subtitle>
                            
<div class="page-header">
    <h2>My title <small>My subtitle</small></h2>
</div>

And should look like:

My title My Subtitle.

Fragment Content basic, no fancy

Fragment content is wrapped with a <content> wrapper. The wrappers can contain any valid markdown. Different Layouts can be achieved using the "type" attribute on the wrapper tag (see: fancy blocks). The wrapper can contain any valid markdown..

This Markdown
Should Generate
<content>
Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
    
Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.

</content>
                        
<div class="row">
    <div class="span16">
        <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
        <p>Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.</p>
    </div>
</div>

And should look like:

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
